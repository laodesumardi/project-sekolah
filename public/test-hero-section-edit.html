Troubleshooting

Jika Form Tidak Muncul:

1. Periksa Database: Pastikan ada data HomepageSetting

2. Clear Cache: php artisan cache:clear

3. Run Seeder: php artisan db:seed --class=HomepageSettingSeeder

4. Check Permissions: Pastikan file bisa diakses

5. Browser Cache: Hard refresh (Ctrl+F5)

URL yang Benar:

Admin Panel: http://127.0.0.1:8000/admin/homepage-settings/edit

Hero Section: http://127.0.0.1:8000/admin/homepage-settings/edit#hero-section

Langkah Verifikasi

Cara Test Hero Section Edit:

1. Buka Admin Panel: Akses URL admin homepage settings

2. Scroll ke Hero Section: Cari bagian dengan ID "hero-section"

3. Isi Form: Masukkan data di field yang tersedia

4. Upload Gambar: Upload background image jika diperlukan

5. Simpan: Klik tombol "Simpan Perubahan"

6. Verifikasi: Lihat hasil di halaman utama
